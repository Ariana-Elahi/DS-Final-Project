🌤️ Welcome to WeatherBot!

This chatbot can answer questions using:

  • 🔵 Real-time weather data from a live API
  • 🟡 Historical weather data from March 2016 (CSV file)

❓ Try Asking:

Send a POST request to /chat with a JSON body like:

{ "message": "current weather in Virginia" }
{ "message": "humidity in Florida" }
{ "message": "UV index in Arizona" }
{ "message": "temperature in Texas in March 2016" }
{ "message": "wind direction in Illinois in 2016" }
